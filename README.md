# **Better Pebble Emulator**

By Only-Meeps

**UBUNTU AND DEBIAN PLATFORMS ONLY**

## **Features**

-faketime built in (easy to change the time for pebble watchfaces)

-pebble build & pebble install in emulator with one command

## **Installation**

1. Download the script from the releases page
2. Run the script from anywhere on your system
3. Type y
4. Tollow prompts to install faketime and the pebble SDK (if you have not yet)
5. Yay, your done!
