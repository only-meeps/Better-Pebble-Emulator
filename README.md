# **Better Pebble Emulator**

By Only-Meeps

**UBUNTU AND DEBIAN PLATFORMS ONLY**

## **Features**

-faketime built in (easy to change the time for pebble watchfaces)

-pebble build & pebble install in emulator with one command

## **Installation**

1. Run the script from anywhere on your system
2. type y
3. follow prompts to install faketime and the pebble SDK (if you have not yet)
4. Yay, your done!
